Basic Usage

Just enter your logo code in the textbox and click the Enter button to execute. Click the little arrow next to the textbox to switch to multiline mode.

Shortcuts

  • ENTER / Ctrl + ENTER (Ctrl in multiline mode)
    Execute the code in the text box
  • Alt + UP / Alt + DOWN
    Execute the code in the text box
  • UP / DOWN / Ctrl + UP / Ctrl + DOWN (Ctrl in multiline mode)
    Move through your previous commands

Commands (case insensitive)

  • FORWARD steps / FD steps
    Move the turtle forward
  • BACK steps / BK steps
    Move the turtle backward
  • RIGHT angle / RT angle
    Turn the turtle right by a given angle
  • LEFT angle / LT angle
    Turn the turtle left by a given angle
  • REPEAT num [ statements ]
    Repeat the statements for a given number of times. Statements can be any combination of above commands or repeat itself.
    For example: REPEAT 4 [ FD 100 RT 90 ] - will draw a square
  • CLEAR / CLS
    Clears the screen
  • COLOR value / CLR val
    Change the color to the given value. Value can be between 1 - 10
  • PENUP
    Lift the turtle's pen so it can move around without drawing
  • PENDOWN
    Put the turtle's pen down
  • HOME
    Send the turtle to the center

Procedures

A procedure can be define by using a define statement and ended using an end. The body of the procedure can contain any number of statements. The procedure name has to be alphabetic and cannot be one of the commands above. 

define sq
fd 50
rt 90
repeat 3 [ fd 50 rt 90 ]
end

To use a procedure, just use the name as part of your statement.

repeat 4 [ fd 25 rt 90 sq ]

Variables

You can declare a variable using a simple assignment statement.

x = 10

And then use them anywhere in the code alongwith with add/multiple operations on them.

clear
x = 5
repeat 12 [ repeat 4 [ fd x rt 90 ] penup lt x fd (x*2) x = x + 5 pendown ]
home
x = 5
repeat 12 [ repeat 4 [ fd x lt 90 ] penup rt x fd (x*2) x = x + 5 pendown ]
